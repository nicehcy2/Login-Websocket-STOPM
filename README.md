# 채팅 기능 개선을 위한 백엔드 서버

## 1. 개요
WebSocket, RabbitMQ, Spring STOMP를 활용한 채팅 애플리케이션 개발 및 연습 저장소.

Redis와 MySQL을 저장소로 사용.

데이터 정합성 문제 및 트랜잭션 불일치 문제를 해결하기 위해 Kafka, Debezium 기반 Outbox 패턴 도입.
